💡 HINT 1 — Halving the Problem
Instead of multiplying x by itself n times (O(n)), what if you split n in half each step?

x^n = x^(n/2) * x^(n/2)

Each step halves n — so you only need ~log₂(n) steps total.
💡 HINT 2 — Handle Odd Exponents
If n is even: x^n = half * half
If n is odd: x^n = x * half * half

Detect odd with bitwise AND: if (n & 1) — much faster than n % 2.
💡 HINT 3 — Go Iterative for Full Marks
Start with result = 1. Loop while n > 0:
  • If n & 1 → result *= x
  • Then x *= x and n >>= 1

This avoids recursion stack and runs in O(log n) time + O(1) space = full 15 pts.
