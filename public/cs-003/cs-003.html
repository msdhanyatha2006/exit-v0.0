#CS-003Hard
The Infinite Loop ⛽
GreedyArray Single PassO(n) Required 30 ptsC (C11)

There are n charging stations arranged in a circular route. Station i provides gas[i] energy. Travelling from station i to i+1 costs cost[i] energy.

You start with an empty tank. Find the starting station index that lets you complete one full clockwise circle without running out of energy. If no such station exists, return -1.

The answer, if it exists, is guaranteed to be unique.

Function Signature
int canCompleteCircuit( int* gas, int* cost, int n);
Example
gas :[1, 2, 3, 4, 5]
cost:[3, 4, 5, 1, 2]
Output:3
Start at index 3 (gas=4). Move to 4: costs 1, refill 5 → tank=8.
Move to 0: costs 2, refill 1 → tank=7. Continue through 1, 2 — just enough to reach 3 again. ✓
Walkthrough from Index 3
1
At station 3: fill 4. Tank = 4
2
Drive to 4 (cost 1). Refill 5. Tank = 4−1+5 = 8
3
Drive to 0 (cost 2). Refill 1. Tank = 8−2+1 = 7
4
Drive to 1 (cost 3). Refill 2. Tank = 7−3+2 = 6
5
Drive to 2 (cost 4). Refill 3. Tank = 6−4+3 = 5
6
Drive back to 3 (cost 5). Tank = 5−5 = 0 ✓ Complete!
Constraints
  • 1 ≤ n ≤ 10⁵
  • 0 ≤ gas[i], cost[i] ≤ 10⁴
  • Return -1 if impossible
  • O(n) time required for full marks

💡 HINT 1 — The "Impossible" Check
Before you even start simulating, think about the total gas vs. the total cost. If the sum of all gas at every station is less than the sum of all costs, is it even possible to finish the circuit from any station?
Mathematical Logic: If Σ gas[i] < Σ cost[i], return -1 immediately.
💡 HINT 2 — The "Broken Bridge" Concept
If you start at station A and run out of gas at station B, don't bother trying to start at any station between A and B. None of those stations will have enough "surplus" gas to get you past station B either.
Action: If current_gas drops below zero at index i,
your next potential starting point should be i + 1.
💡 HINT 3 — The Greedy Optimization (The O(n) Secret)
You only need one pass through the array. Keep track of two things:
Total Surplus : overall balance of (Gas − Cost) for the entire trip.
Current Tank : your gas balance since your last attempted starting point.

If current_tank hits a negative number →
  reset it to zero and set the next station as your new starting candidate.

If total_surplus ≥ 0 at the end → your last candidate is the winner.
